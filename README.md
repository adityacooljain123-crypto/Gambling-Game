# Gambling-Game 🎰
This is my first game and project I have built. For context, I am a new creator who started out just 4 months ago and am eager to make and learn about coding and all the great things it can do. 

Usage instructions 🧾:

  For this program, you only need python (ideally the latest version) and the two files (Gambling.py and blacklist.txt) located in the branches where the README file is to play. To launch the game, just double click on the python file and run with python (default).

Features 🎯:
1. Random Money Generator
2. 3 mini games which are Battleships, Dice, and Race.
3. A dealer with a short temper
4. A blacklist

Games 👾:
1. 🏁 Race: You will bet on a number of racers to see who will win first. In my opinion the easiest to win as there is only 6 racers. Its like horse racing, but digital
2. 🎲 Dice: You first choose from one of two gamemodes, 1. Guess the numbers, or 2. Odds or Evens.

       🔢  Guess the number: The dealer chooses a number at random and you will have to guess it within a few attempts.
            Win and recieve a big prize but lose and lose your funds

       🪙   Odds or Evens: There is a six-sided dice where you will bet on whether the number it lands on is odd or even.
             Its like a coin flip.

4. 🛳️ Battleships: There will be a hidden 8 x 8 grid where the dealer chooses three random squares to place his ships. Your goal is to destroy all his ships within a limited amount of turns

If you do make the dealer angry, he will kick you out. So be careful

This is my first game so I hope you enjoy it and please give me ideas for adding things. I will be actively improving and updating this project.
